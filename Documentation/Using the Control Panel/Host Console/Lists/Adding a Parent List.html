Adding a Parent List

How to add a parent list that will be available to all sites using the Host Lists module. This tutorial shows how to add a Category list that can be used for the Documents module.

  1. Navigate to Host > Lists.
  2. Click the Add List link.

  1. In the List Name text box, enter a name for the list. E.g. Documents
  2. At Parent List, leave this field set to None Specified.
  3. In the Entry Text text box, enter the first entry (item) that will be in this list. E.g. Forms
  4. In the Entry Value text box, enter the identifier or code for the first entry. E.g. FM or Form. Note: When creating a categories list for the Documents module, the Entry Value is displayed as the category name.
  5. Optional. At Enable Sort Order, select from these options:
    • if you want to be able to reorder the entries in this list.
    • to use alphabetical sort order for list entries.

  1. Click the Save link.
  2. The newly added parent list is now displayed, enabling you to commence adding list entries to this list.
