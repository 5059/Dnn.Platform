Deleting a List

How to delete a list from the Admin Lists page. Note: Only user generated lists can be deleted.

  1. Navigate to Admin > Advanced Settings > Lists.
  2. In the Lists section, navigate to and select the title of the required list. E.g. Country.Australia:Region. This displays details of this list and its related list entries.
  3. Click the Delete button. This displays the message "Are You Sure You Wish To Delete This Item?"
